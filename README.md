# 🎓 NexusERP

**NexusERP** is a modern **School & College ERP (Enterprise Resource Planning) platform** built with **React, TanStack Start, TypeScript, Tailwind CSS, Supabase, and Cloudflare Workers**.

The platform helps educational institutions manage **students, faculty, departments, attendance, examinations, fees, library operations, timetables, announcements, and campus events** through a centralized dashboard.

NexusERP simplifies academic and administrative operations by providing a secure, scalable, and user-friendly management system for schools, colleges, and universities.

# ⚙️ Environment Variables

Create a `.env` file in the project root:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```

# ▶️ Run Locally

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/nexuserp.git
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Start Development Server

```bash
npm run dev
```

Application runs at:

```
http://localhost:3000
```
